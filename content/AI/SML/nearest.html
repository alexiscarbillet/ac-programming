Nearest Neighbors

The supervised nearest neighbors algorithm is a machine learning algorithm that helps us make predictions or classify new data points based on the data we already have.

Here's how it works in simple terms:

1. Let's say we have a dataset with many examples, and each example has a set of features and a corresponding label or category. For example, we have a dataset of fruits where each fruit has features like color, shape, and size, and a label indicating whether it's an apple or an orange.

2. When we want to predict or classify a new fruit, we find the k nearest neighbors to that fruit from our existing dataset. The "k" is a parameter we set and determines how many neighbors we consider.

3. To find the nearest neighbors, we calculate the distance between the features of the new fruit and the features of all the fruits in our dataset. Common distance measures include Euclidean distance or Manhattan distance. The fruit with the smallest distance to the new fruit is considered the closest neighbor.

4. Once we have found the k nearest neighbors, we look at their labels. If we are predicting a label, we can take a majority vote among the neighbors' labels and assign the most common label to the new fruit. If we are performing regression (predicting a continuous value), we can take the average of the neighbors' values.

5. Finally, we have our prediction or classification for the new fruit based on the labels of its nearest neighbors.

The idea behind the supervised nearest neighbors algorithm is that similar examples tend to have similar labels or values. By finding the nearest neighbors in our dataset, we can make predictions for new examples based on the known examples we have. It's a simple yet powerful algorithm for various machine learning tasks.

Python Example

from sklearn.neighbors import KNeighborsClassifier

# Creating the dataset
X = [[2.0, 4.0], [4.0, 6.0], [1.5, 3.5], [3.5, 5.5]]  # Features (fruit attributes)
y = ['apple', 'apple', 'orange', 'orange']  # Labels (fruit types)

# Creating the nearest neighbors classifier
k = 3  # Number of neighbors to consider
knn = KNeighborsClassifier(n_neighbors=k)

# Training the classifier
knn.fit(X, y)

# Making predictions for new fruits
new_fruits = [[3.0, 5.0], [2.5, 4.5]]  # New fruits' features
predictions = knn.predict(new_fruits)

# Printing the predictions
for i, fruit in enumerate(predictions):
  print("New fruit", i+1, "is predicted to be:", fruit)

Results:

New fruit 1 is predicted to be: apple
New fruit 2 is predicted to be: orange

In this example, we have a dataset with four fruits, each with two features: X-coordinate and Y-coordinate. The fruits are labeled as either 'apple' or 'orange'. We want to classify two new fruits based on their coordinates.

We create an instance of the KNeighborsClassifier class, set the number of neighbors (k) to 3, and then train the classifier using the fit method with our features X and labels y.

Next, we define the features of two new fruits in the new_fruits list. We use the predict method of the classifier to make predictions for these new fruits, and store the results in the predictions variable.

Finally, we print the predicted labels for the new fruits. The output will display the predicted fruit type for each new fruit based on its nearest neighbors in the dataset.

Note: This is a simplified example for illustrative purposes. In practice, you would typically have a larger dataset with more features and more complex tasks.
