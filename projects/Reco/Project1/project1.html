Movie recommender

This movie recommender system is based on this dataset. This project works offline, you will need to have a Python IDE in order to run it. It contains also a script used in order to download the images of each movie. This one can be very long because of the large amount of data. You can find the code here

The recommender used depends on the input. There are 3 variables: the last movie watched, the user ID and the genres (20 different categories). According to the combination of those paramaters, a different recommender is used. You can find here which recommender is used according to the input:

VariablesRecommender
Last movie watchedmetadata recommender
User IDSVD
genresmetadata recommender
Last movie watched & genresmetadata recommender
Last movie watched & User IDSVD
User ID & genresNone
Last movie watched & User ID & genresSVD

Here is the interface before any research. As you can see, many checkboxes are present, one per genre. The movies recommended are displayed with their image, their title, their year, their rating, their genres and their description. Only the top 5 movies are displayed.
